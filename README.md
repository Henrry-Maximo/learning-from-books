## Como está organizado?

A estrutura do repositório é dividida por livro dentro da pasta `/books`. Cada livro possui:
- `anotacoes.md`: Resumos conceituais, mapas mentais textuais e insights gerados durante a leitura.
- `/exercicios`: Códigos e desafios propostos pelo livro replicados na prática.
